```
基于 MCP 协议的邮件服务器，支持 SMTP/IMAP/POP3 协议，可发送和接收邮件。支持 Gmail、Outlook、QQ、163、126、Sina 等主流邮箱服务商，支持附件、HTML/文本格式，动态认证。完美集成 Cursor、Claude Desktop 等 AI 助手，实现邮件自动化处理。
```

## 更新步骤

1. 访问仓库页面：https://github.com/1018053166/sse-email-mcp-server
2. 点击仓库右侧的 **Settings** 按钮
3. 滚动到 **About** 部分
4. 在 **Description** 中输入上面的中文描述
5. 在 **Topics** 中添加上面的主题标签（每行一个或逗号分隔）
6. 点击 **Save changes** 保存

## 工具能力描述

### 核心功能
- 📧 **邮件发送** - 通过 SMTP 协议发送邮件，支持 HTML 和纯文本格式
- 📥 **邮件接收** - 通过 IMAP/POP3 协议接收邮件，支持查询未读邮件
- 📎 **附件支持** - 支持发送文件附件和 Base64 编码附件
- 🔐 **安全认证** - 支持动态传递认证信息，更安全

### 支持的邮箱服务商
- Gmail
- Outlook/Office365
- QQ 邮箱
- 163 邮箱
- 126 邮箱
- Sina 邮箱
- 自定义邮箱服务器

### 使用场景
- AI 助手集成（Cursor、Claude Desktop）
- 开发与测试（邮件通知、测试邮件）
- 工作流自动化（定时发送、邮件归档）
- 企业应用（客户服务、系统告警、报告生成）
